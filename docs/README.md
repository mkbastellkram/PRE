# PRX Map Only · Madeira

Minimalmodul zum isolierten Test der Leaflet-Karte.

Enthalten:
- Madeira-Fullscreen-Karte
- OSM Standard
- OpenTopoMap
- Esri Satellit
- Esri Satellit + Carto Label Overlay als Hybrid
- keine Pins
- keine PR-Daten
- keine GPX/KML
- kein Audit
- kein Service Worker

Ziel: Beweisen, dass die Kartenbasis isoliert stabil läuft, bevor PRX-Funktionen integriert werden.
